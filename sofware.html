software

1. El software de gestión de bases de datos (DMS), permitira a la escuela nacional de la calidad del cafe mejorar la accesibilidad a los datos y simplifica el proceso de gestión.

2. Alojamiento web: Necesitará un alojamiento web para poder funcionar, por lo que se requerirá un servicio de alojamiento web. hay que tener en cuenta requiere ciertos requisitos de configuracion, como disponibilidad de base de datos.

3. Dominio:No necesariamente requiere de un dominio, pero es recomendable tener uno para acceder al software a través de la web.

4. SSL: Se necesita un certificado SSL para asegurarse de que la información de los clientes se transmita de manera segura. Muchos proveedores de alojamiento web ofrecen SSL gratuito, aunque también puede comprar uno a través de un proveedor de SSL como GlobalSign, DigiCert, entre otros.
